---

I build probabilistic models and geospatial systems that quantify uncertainty rather than hide it. Interested in Bayesian hierarchical modeling, spatial statistics, and turning complex geographic data into interpretable, interactive dashboards.

- Fitting BYM2 spatial models in **PyMC** across hundreds of census tracts and counties
- Building ETL pipelines and A/B testing frameworks for performance analytics
- Combining **QGIS**, **MapLibre-GL**, and **D3** for spatial storytelling at scale

| Project | Description | Stack |

| [Bayesian Transit Equity](https://bayes-transit-equity.vercel.app) | Spatial hierarchical model across 726 SD census tracts; posterior exceedance analysis + intervention simulation | PyMC, Next.js, Deck.gl |
